# Creative Apps Rules

## Shared Creative App Principles

- Prefer launch via `Win+S`, then verify the window title, canvas, and toolbars before acting.
- Prefer visible side panels, toolbars, ribbons, inspectors, layers panels, timeline panels, and export dialogs over blind clicking on the canvas.
- Treat modal dialogs, export sheets, save prompts, and file pickers as the active surface until they are closed.
- Before export, render, delete, flatten, overwrite, or publish actions, verify the target file/project and requested output settings.

## Adobe Photoshop

- Launch/focus: `Win+S`, type `Photoshop`, press `Enter`.
- Common shortcuts:
  - `Ctrl+O` open
  - `Ctrl+S` save
  - `Ctrl+Shift+S` save copy / alternate save flows
  - `Ctrl+Shift+E` merge visible
  - `Ctrl+T` free transform
  - `V` move tool
  - `B` brush tool
  - `M` marquee tool
  - `L` lasso tool
  - `Ctrl+0` fit on screen
  - `Ctrl+1` 100 percent zoom
- Prefer the Layers panel, toolbar, top options bar, and Properties panel for targeted actions.

## Figma

- Figma is usually browser-based or desktop-app based; verify which surface is active first.
- Common shortcuts:
  - `Ctrl+/` quick actions
  - `V` move tool
  - `T` text tool
  - `R` rectangle
  - `O` ellipse
  - `F` frame tool
  - `Ctrl+D` duplicate
  - `Shift+1` zoom to fit
- Prefer left layers panel, top toolbar, right design/prototype panel, and exact layer names.

## Adobe Premiere Pro

- Launch/focus: `Win+S`, type `Premiere Pro`, press `Enter`.
- Common shortcuts:
  - `Ctrl+O` open project
  - `Ctrl+S` save
  - `Space` play/pause
  - `I` mark in
  - `O` mark out
  - `C` razor tool
  - `V` selection tool
  - `Ctrl+M` export media
- Prefer the Project panel, timeline, Program Monitor, Effects panel, and Export dialog.
- Verify the active sequence and selected clip before edits.

## DaVinci Resolve

- Launch/focus: `Win+S`, type `DaVinci Resolve`, press `Enter`.
- Prefer the page tabs at the bottom: Media, Cut, Edit, Fusion, Color, Fairlight, Deliver.
- Use exact panel names when targeting bins, inspector controls, timeline tracks, or deliver settings.

## Blender

- Launch/focus: `Win+S`, type `Blender`, press `Enter`.
- Common shortcuts:
  - `G` grab/move
  - `R` rotate
  - `S` scale
  - `Tab` object/edit mode toggle
  - `Shift+A` add menu
  - `N` sidebar toggle
  - `Ctrl+S` save
  - `F12` render image
- Prefer the Outliner, Properties editor, viewport gizmos, and top workspace tabs.

## Reliable Creative Workflow Patterns

### Open an existing project
1. Launch or focus the app.
2. Use the app's open shortcut or visible recent-project surface.
3. Verify the project/document name after load.

### Export or render
1. Open the export/render flow.
2. Verify format, destination, and preset/options.
3. Confirm only when the exact output matches the request.

### Edit a layer, clip, or object
1. Identify the correct layer/clip/object first.
2. Verify selection state visually.
3. Apply the edit in the most direct panel or toolbar.
4. Re-check the canvas/timeline/output state after the edit.
